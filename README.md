# arcade-menu

An arcade menu built on top of React and Electron

## Setup

* `yarn`

In two separate terminals, simultaneously run (in order):

* `yarn start`
* `yarn electron`

## Background

Based on [frontend-kit](https://github.com/FruitieX/frontend-kit)
